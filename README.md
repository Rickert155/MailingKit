# Mailing Kit
Набор инструментов для рассылки, начиная от сбора контактных баз до рассылки писем через удаленные сервера или Selenium  

- Парсер email дженерироков по доменам ![link](https://github.com/Rickert155/MailingKit/gerericEmail)
- Рассыльщик писем через формы на сайтах ![link](https://github.com/Rickert155/MailingKit/botForms)
- Рассыльщик писем через веб-интерфейс roundcube ![link](https://github.com/Rickert155/MainlingKit/WebMail)
- Рассыльщик писем через удаленные сервера ![link](https://github.com/Rickert155/MailingKit/Mailing)
